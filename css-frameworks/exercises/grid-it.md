### Challenge: Grid it
1. How much of the page would elements with these classes take up?

  * `col-md-2`
  * `col-sm-1`
  * `col-lg-8`
  * `col-xs-12`


1. Using the bootstrap grid, make a grid that is 3 Columns on Tablet (sm), Laptop (md), and Desktop (lg), 1 Column on Mobile(xs).

</div>
